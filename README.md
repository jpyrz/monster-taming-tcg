# Monster Command TCG

Working prototype for a 1v1 monster battling card game. The repository name is
temporary; the final game title is still undecided.

## Stack

- React
- Vite
- TypeScript
- Mantine
- SCSS modules
- Cypress component tests

## Commands

```bash
npm run dev
npm run build
npm run typecheck
npm run lint
npm run test:component
```
